# Multi-Face Detection App

This Streamlit app allows you to detect multiple faces in images or videos efficiently. It uses a pre-trained face detection model and provides options for image enhancement and face detection.

## Features
- **Efficient Face Detection**: Utilizes a pre-trained face detection model for fast and accurate detection.
- **Image Enhancement**: Allows adjusting brightness, contrast, and sharpness of the image.
- **Face Detection**: Detects multiple faces in the image or video.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/multi-face-detection-app.git
   ```
2. Navigate to the project directory:
   ```
   cd multi-face-detection-app
   ```
3. Create a virtual environment:
   ```
   python -m venv venv
   ```
4. Activate the virtual environment:
   - On Windows:
     ```
     .\venv\Scripts\activate
     ```
   - On macOS and Linux:
     ```
     source venv/bin/activate
     ```
5. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
1. Run the Streamlit app:
   ```
   streamlit run app.py
   ```
